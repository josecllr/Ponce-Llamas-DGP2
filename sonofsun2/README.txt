IMPORTANTE LEER ANTES DE USAR EL PROGRAMA

INSTRUCCIONES GENERALES PARA CREAR UNA ANIMACION POR MEDIO DE ESTE PROGRAMA:

-CREAR UN ARCHIVO PNG DEL DISEÑO DESEADO PARA LA ANIMACIÓN
-GUARDAR EL ARCHIVO EN LA CARPETA DEL PROGRAMA (SONOFSUN2) CON EL NOMBRE DE "cuerpo2.png"
-EJECUTAR EL PROGRAMA
-USAR LAS TECLAS LISTADAS ABAJO PARA MODIFICAR LA ANIMACION
-IMPORTAR LAS IMAGENES A SU EDITOR DE VIDEO PREDETERMINADO
-USAR CHROMA KEY PARA BORRAR EL FONDO
-COMPONER UNA ANIMACIÓN
-EXPORTARLA COMO VIDEO

***IMPORTANTE***
[ACTIVAR SAVEFRAME NOLOOP (LETRA L) AL FINAL DEL CODIGO PARA GUARDAR LOS FOTOGRAMAS]
